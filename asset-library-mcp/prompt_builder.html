🧭 Yön Kalibrasyonu

Her asset'in varsayılan yönü farklı olabilir. Test edip düzeltin.

Nasıl kalibre edilir:

1. "Blender'da... Aktar" → obje origin'e gelir (rot=0)

2. Numpad 7 üstten bak

3. "Ön yüz" nereye bakıyor?

−Y'e (sana doğru) → güney → correction = 0

+Y'e → kuzey → correction = 180

+X'e → doğu → correction = 270

−X'e → batı → correction = 90
